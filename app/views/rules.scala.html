Die Regeln des Roulette

Grundlagen

Roulette ist ein klassisches Casinospiel, bei dem eine kleine Kugel in ein sich drehendes Rad geworfen wird. Spieler setzen darauf, wo die Kugel landen wird.

Das Roulette-Rad

Das europäische Roulette-Rad hat 37 Zahlen von 0 bis 36. Die amerikanische Variante hat zusätzlich eine doppelte Null (00).

Wettmöglichkeiten

  • Einzelne Zahlen (Straight Up): Setzen Sie auf eine einzelne Zahl.
  • Rot oder Schwarz: Setzen Sie auf die Farbe, in der die Kugel landen wird.
  • Gerade oder Ungerade: Setzen Sie darauf, ob die Zahl gerade oder ungerade sein wird.
  • Hoch oder Niedrig: Setzen Sie auf die Zahlen 1-18 (niedrig) oder 19-36 (hoch).

Spielablauf

  1. Spieler platzieren ihre Einsätze auf dem Tisch.
  2. Der Croupier wirft die Kugel in das sich drehende Rad.
  3. Wenn die Kugel zur Ruhe kommt, wird die Gewinnzahl und -farbe bekannt gegeben.
  4. Gewinnende Einsätze werden ausbezahlt, verlorene Einsätze werden eingezogen.

Viel Glück und spielen Sie verantwortungsvoll!
